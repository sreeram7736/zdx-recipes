<h1 align='center'>ZDX Recipe</a></h1>

<h4>ZDX is the emerging cinematic framework for creating servers on FiveM, with many official and community resources designed to utilise the tools provided here.</h4>

For a taste of what's available:
<div><ol type=dot>
<li>ZDX_ShadowCut:- Shadow Cut records your vehicle's movements and allows you to record and film against a real-time ghost of your previous run.</li>
<li>ZDX_SpiderCam:- Spider Cam allows players to record gameplay and create cinematic scenes using advanced free-camera controls and custom angles.</li>
<li>ZDX_scenedirector:-Scene Director lets players create and control dynamic GTA V scenes by directing multiple NPCs, vehicles, animations, tasks, and camera shots in real time   for machinima and cinematic productions.</li>
<li>ZDX_loadingscreen:- ZDX Loading Screen provides a customizable server introduction experience featuring music, server information, rules, staff details, progress indicators, and branding while players connect to a FiveM server.</li>
</ol>
</div>
Many more resources are included in this Recipe, or you can browse the Cfx.re Releases board for more.
